💡 학습 포인트 — Step 4의 핵심: 데이터 시각화

PRD에 "차트로 보여줘"만 추가하면, AI가 Chart.js 라이브러리를 활용해 자동 생성합니다.
숫자 나열 → 시각적 인사이트. 보고용 자료로도 바로 활용 가능합니다.

💡 학습 포인트 — Chart.js란?
Chart.js는 웹에서 차트를 그리는 무료 라이브러리입니다.
CDN 한 줄로 불러오고, JavaScript로 데이터만 넣으면 자동으로 차트가 그려집니다.
// 1. CDN으로 Chart.js 불러오기 (HTML <head>에 추가) <script src="https://cdnjs.cloudflare.com/ajax/libs/Chart.js/4.4.0/chart.umd.min.js"> // 2. 도넛 차트 생성 — 데이터만 넣으면 됨 new Chart(document.getElementById('typeChart'), { type: 'doughnut', // ← 차트 종류 (doughnut, bar, line, pie...) data: { labels: ['가격 담합', '소비자 피해', '허위광고', ...], // ← 항목 이름 datasets: [{ data: [45, 38, 32, ...], // ← 숫자 데이터 backgroundColor: ['#1a3a6b', '#2a5298', '#3a7bd5', ...] }] } }); // 🔧 PRD 수정 팁: // "파이 차트로 바꿔줘" → type: 'pie' // "선 그래프로 바꿔줘" → type: 'line' // "3D 느낌으로 해줘" → 그림자 효과 추가
