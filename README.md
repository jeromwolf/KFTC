# 공정거래위원회 바이브코딩 교육 프로젝트 (KFTC)

공정거래위원회 직원 대상 **바이브코딩(Vibe Coding)** 교육을 위한 실습 프로젝트입니다.
AI 도구(안티그래비티/Claude)를 활용하여 **PRD 한 장으로 업무용 웹앱을 만드는 과정**을 체험합니다.

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 대상 | 공정거래위원회 직원 (비개발자) |
| 목적 | AI 코딩 도구를 활용한 업무 자동화 체험 |
| 도구 | 안티그래비티 (Antigravity) |
| 기술 | 순수 HTML/CSS/JS (빌드 도구 없음, 브라우저에서 바로 실행) |

---

## 교육 진행 방식

### Step 1~5: 점진적 빌드 (민원 AI 분류 대시보드)

PRD 한 장에서 시작하여, 채팅으로 기능을 하나씩 추가해가며 완성도를 높이는 과정을 체험합니다.

| Step | 핵심 키워드 | 방식 | 설명 |
|------|-----------|------|------|
| **Step 1** | 엑셀 업로드 | **PRD** (`step1/PRD.md`) | 엑셀 파일을 테이블로 표시하는 기본 앱 |
| **Step 2** | 디자인 | 채팅 프롬프트 | 공공기관 파란 테마 + 통계 카드 UI |
| **Step 3** | AI 분류 | 채팅 프롬프트 | Claude API로 민원 자동 분류 (유형/부서/우선순위) |
| **Step 4** | 시각화 | 채팅 프롬프트 | Chart.js 도넛 차트 + 바 차트 |
| **Step 5** | 완성본 | 채팅 프롬프트 | 필터, 엑셀 다운로드, 인쇄 기능 |

### 템플릿 4종: 업무 문서 자동 생성

실제 공정거래위원회 업무에서 사용할 수 있는 문서 생성 도구입니다.
모두 **정부 표준 공문 양식**으로 출력되며, **PDF 다운로드**를 지원합니다.

| 템플릿 | 방식 | 설명 |
|--------|------|------|
| **출장결과보고서** | 채팅 프롬프트 | 출장 정보 입력 → 보고서 자동 작성 |
| **민원답변공문** | 채팅 프롬프트 | 민원 내용/처리결과 → 답변 공문 생성 |
| **회의록자동구조화** | 채팅 프롬프트 | 자유 메모 → 안건/결정/후속조치 구조화 |
| **공문안내문생성기** | 채팅 프롬프트 | 유형별 공문/안내문 자동 생성 |

## 기술 스택

| 기술 | 용도 | CDN |
|------|------|-----|
| **SheetJS** (xlsx.js) | 엑셀 파일 파싱/생성 | `cdnjs.cloudflare.com/ajax/libs/xlsx/0.18.5/xlsx.full.min.js` |
| **Chart.js** | 도넛/바 차트 시각화 | `cdnjs.cloudflare.com/ajax/libs/Chart.js/4.4.0/chart.umd.min.js` |
| **html2pdf.js** | HTML → PDF 변환 | `cdnjs.cloudflare.com/ajax/libs/html2pdf.js/0.10.1/html2pdf.bundle.min.js` |
| **Claude API** | AI 분류/문서 작성 | `api.anthropic.com/v1/messages` |

### Claude API 사용 방식

브라우저에서 직접 Claude API를 호출합니다 (서버 없음):

```javascript
fetch('https://api.anthropic.com/v1/messages', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': apiKey,
    'anthropic-version': '2023-06-01',
    'anthropic-dangerous-direct-browser-access': 'true'  // 브라우저 직접 호출 허용
  },
  body: JSON.stringify({
    model: 'claude-sonnet-4-5-20250929',
    max_tokens: 2000,
    system: systemPrompt,
    messages: [{ role: 'user', content: userContent }]
  })
});
```

### 두 가지 동작 모드

모든 AI 연동 페이지는 API 키 유무에 따라 자동 전환됩니다:

| 모드 | 조건 | 동작 |
|------|------|------|
| **AI 모드** | API 키 있음 | Claude가 내용을 분석하고 격식체로 작성 |
| **기본 모드** | API 키 없음 | 템플릿/키워드 기반으로 자동 생성 |

## 실행 방법

### 1. 파일 열기

별도 설치가 필요 없습니다. HTML 파일을 브라우저에서 직접 열면 됩니다.

```
step1/index.html    → 더블클릭으로 열기
step5/index.html    → 더블클릭으로 열기
templates/출장결과보고서.html → 더블클릭으로 열기
```

### 2. API 키 설정

**방법 A — config.js (교육 배포용)**

`config.js` 파일에 API 키가 포함되어 있으면 자동으로 로드됩니다.
이 파일은 `.gitignore`에 포함되어 GitHub에는 올라가지 않습니다.

```javascript
// config.js
const KFTC_CONFIG = {
  API_KEY: 'sk-ant-...',
  MODEL: 'claude-sonnet-4-5-20250929'
};
```

**방법 B — 수동 입력**

각 페이지의 API 키 입력창에 직접 입력하면 `localStorage`에 저장되어
다른 페이지에서도 자동 적용됩니다.

### 3. 샘플 데이터

`sample_민원목록_200건.xlsx` 파일을 Step 1~5에서 업로드하여 사용합니다.

## 교육 프롬프트 가이드

`교육진행_프롬프트_가이드.md` 파일에 안티그래비티 채팅창에 넣을 프롬프트가 정리되어 있습니다.

**교육 순서:**

1. **Step 1** — `step1/PRD.md` 내용을 안티그래비티에 붙여넣기
2. **Step 2** — "디자인 업그레이드해줘" 프롬프트 입력
3. **Step 3** — "AI 분류 추가해줘" 프롬프트 입력
4. **Step 4** — "차트 추가해줘" 프롬프트 입력
5. **Step 5** — "필터/다운로드/인쇄 추가해줘" 프롬프트 입력
6. **템플릿** — 각각 별도 프롬프트로 새 파일 생성

## 배포 방법

### GitHub (코드 공유용)

```bash
git clone https://github.com/jeromwolf/KFTC.git
```

> `config.js`는 포함되지 않습니다. API 키는 별도 전달합니다.

### 교육 현장 배포 (USB/압축파일)

프로젝트 폴더를 통째로 복사하면 됩니다.
`config.js`에 API 키가 포함되어 있어 별도 설정 없이 바로 사용 가능합니다.

```
KFTC/
├── config.js              ← API 키 포함
├── sample_민원목록_200건.xlsx
├── step1/ ~ step5/
└── templates/
```

---

## 주의사항

- `config.js`의 API 키는 **교육용**입니다. 교육 종료 후 삭제됩니다.
- 브라우저에서 Claude API를 직접 호출하므로 **인터넷 연결이 필요**합니다.
- API 키 없이도 **기본 모드**로 동작하며, 핵심 기능은 체험 가능합니다.
- 모든 파일은 **단일 HTML** 파일로, 별도 서버나 설치가 필요 없습니다.
