💡 학습 포인트 — 엑셀 파일은 어떻게 읽을까?
브라우저는 기본적으로 엑셀 파일을 읽을 수 없습니다.
SheetJS (xlsx.js) 라이브러리를 CDN으로 불러와서 엑셀을 JavaScript 배열로 변환합니다.
// 1. 사용자가 파일을 선택하면 fileInput.addEventListener('change', handleFile); // 2. FileReader로 파일을 읽고 const reader = new FileReader(); reader.readAsArrayBuffer(file); // 3. SheetJS가 엑셀 → JSON으로 변환 const workbook = XLSX.read(data, { type: 'array' }); const sheet = workbook.Sheets[workbook.SheetNames[0]]; const json = XLSX.utils.sheet_to_json(sheet); // → [{번호:1, 제목:"...", 민원내용:"..."}, ...]
📋 총 민원: 0건
💡 학습 포인트 — 데이터를 화면에 표시하기
엑셀에서 읽은 JSON 배열을 forEach로 반복하면서
HTML <tr><td> 태그로 변환하여 테이블에 삽입합니다.
// JSON 배열 → HTML 테이블 행으로 변환 data.forEach(row => { const tr = `<tr> <td>${row.번호}</td> <td>${row.제목}</td> <td>${row.민원내용}</td> <td>${row.접수일}</td> </tr>`; tbody.innerHTML += tr; });
번호 제목 민원내용 접수일
