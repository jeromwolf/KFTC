💡 학습 포인트 — AI API 호출은 이렇게 동작합니다
① 민원 1건의 제목+내용을 Claude에게 전송
② 시스템 프롬프트가 "JSON으로만 답해"라고 지시
③ Claude가 {"유형":"가격 담합", "담당부서":"카르텔조사국", ...} 형태로 응답
④ 프로그램이 JSON을 파싱하여 테이블에 표시
// 브라우저에서 직접 Claude API를 호출하는 코드 const response = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-api-key': apiKey, // ← 발급받은 API 키 'anthropic-version': '2023-06-01', // ← API 버전 'anthropic-dangerous-direct-browser-access': 'true' // ← 브라우저 직접 호출 허용 }, body: JSON.stringify({ model: 'claude-sonnet-4-5-20250929', // ← 사용할 AI 모델 max_tokens: 1000, // ← 최대 응답 길이 system: '당신은 민원 분류 전문가...', // ← 시스템 프롬프트 (AI의 역할) messages: [{ role: 'user', content: '민원 내용: ...' }] }) }); // 응답 예시: // {"유형":"가격 담합","담당부서":"카르텔조사국","우선순위":"긴급", // "요약":"입찰 담합 의혹 제기","관련법령":"공정거래법 제40조"}
💡 학습 포인트 — 프롬프트 엔지니어링이란?
AI에게 "JSON으로만 응답해"라고 지시하면 → 프로그램이 파싱 가능한 데이터를 받습니다.
이것이 프롬프트 엔지니어링의 핵심입니다. 프롬프트를 바꾸면 결과가 바뀝니다.
// 이 프롬프트가 AI의 응답 형식을 결정합니다 "당신은 공정거래위원회 민원 분류 전문가입니다. 민원 내용을 분석하여 아래 JSON 형식으로만 응답하세요: { \"유형\": \"가격 담합|불공정 거래행위|허위·과장 광고|...\", \"담당부서\": \"카르텔조사국|시장감시국|소비자정책국|...\", \"우선순위\": \"긴급|보통|낮음\", \"요약\": \"20자 이내 핵심 요약\", \"관련법령\": \"공정거래법 제OO조|...\" }" // 🔧 PRD 수정으로 바꿀 수 있는 것들: // "유형에 '개인정보 침해'를 추가해줘" → 카테고리 추가됨 // "우선순위를 4단계로 바꿔줘" → 긴급/높음/보통/낮음 // "감정 분석도 추가해줘" → 새 분류 항목 생성 // "요약을 50자로 늘려줘" → 요약이 더 상세해짐
