💡 학습 포인트 — AI가 만든 CSS 변수란?
--primary: #1a3a6b 하나를 바꾸면 헤더, 버튼, 테이블 색상이 한번에 변경됩니다.
AI는 이런 변수 시스템을 자동으로 설계합니다.
/* AI가 생성한 CSS 변수 — 한 곳만 바꾸면 전체가 바뀜 */ :root { --primary: #1a3a6b; /* 이 값을 #006400 으로 바꾸면? → 초록 테마! */ --primary-light: #2a5298; --bg: #eef2f7; /* 배경색 */ --white: #ffffff; --border: #d0d9e8; } /* PRD 수정 팁: "빨간색 테마로 바꿔줘" → --primary: #8b0000 "초록색 테마로 바꿔줘" → --primary: #006400 "다크모드로 바꿔줘" → --bg: #1a1a2e, --text: #eee */
